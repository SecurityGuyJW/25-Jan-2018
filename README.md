# Project Title

This program sends a file. 

## Getting Started

1. Download code

2. Give the script executable rights

3. type sendFile.py -t (your IP) -p (your port) -f (your file dir)

   a. exmaple -t 192.168.1.1 -p 5000 -f C://hello/notepad.txt

### Prerequisites

Python 2.4 and above
